# Vip-Center-Backend

Backend básico para gestionar un gimnasio Vip-Center.

## Ejecutar proyecto

```bash
npm install
npm start
```

Servidor por defecto: `http://localhost:3000`

## Endpoints disponibles

- `GET /api/health`
- `GET /api/plans`
- `GET /api/members`
- `POST /api/members`
- `GET /api/checkins`
- `POST /api/checkins`
